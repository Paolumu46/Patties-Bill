# ระบบจัดการบิล Patties Bill (Modern Web App)

ระบบใบแจ้งหนี้เวอร์ชันนี้ได้รับการอัปเกรดให้เป็น **Web Application แบบหน้าเดียว (SPA - Single Page Application)** ที่สามารถจัดการบิล และ**ทำตัวเป็น File Manager สำหรับ Google Drive** ได้ในตัว!

## ฟังก์ชันเด่นที่เพิ่มเข้ามา (Features)
- 💾 **ระบบจัดการไฟล์ PDF (Google Drive Integration):**
  - คุณสามารถกดปุ่มเดียวเพื่อสร้างและบันทึกบิลที่แก้ไขเสร็จแล้ว เป็นไฟล์ **PDF** ลงใน Google Drive โดยตรง
  - มีหน้า "จัดการไฟล์ PDF" เพื่อดูลิสต์ไฟล์ PDF ของบิลทั้งหมดในโฟลเดอร์ สามารถ **กดดู (Preview)** หรือ **ลบ (Delete)** ลงถังขยะ Google Drive ได้จากหน้าเว็บโดยตรง
- 🔍 **ระบบค้นหา (Search):** ค้นหาบิลได้อย่างรวดเร็ว จากชื่อลูกค้า, เบอร์โทร, ห้อง หรือ Log No.
- 📝 **เพิ่ม/ลบ/แก้ไข (CRUD):** ไม่ต้องไปแก้ใน Google Sheet ด้วยตัวเองอีกต่อไป สามารถจัดการบิลได้จากหน้าเว็บ
- 🧮 **คำนวณและแปลงยอดเงินอัตโนมัติ:** เมื่อกรอกตัวเลขค่าเช่า ระบบจะรวมยอด และ *แปลงตัวเลขนั้นเป็นตัวอักษรภาษาไทย* ให้อัตโนมัติ
- 🖨️ **รูปแบบบิลที่เป๊ะเหมือนเดิม:** เมื่อกดพิมพ์หน้ากระดาษ (Print) จะเข้าสู่เทมเพลตบิลแบ่งครึ่งสวยงาม (A17/A18) ตามต้นฉบับ 100%

## 🚀 วิธีการติดตั้งและตั้งค่า Google Drive

### ขั้นตอนที่ 1: เตรียม Google Sheets (Database)
1. สร้าง/เปิด Google Sheet เปล่าๆ แล้วสร้าง Sheet ใหม่ให้ชื่อว่า `Settings` เอาไว้ด้วย (ระบบจะเอาไว้เก็บว่าต้องเซฟ PDF ไปลงโฟลเดอร์ไหน)
2. หรือถ้าระบบไม่เจอเดี๋ยวสคริปต์จะสร้างให้ล่วงหน้าเมื่อรันครั้งแรก

### ขั้นตอนที่ 2: เตรียมโฟลเดอร์ใน Google Drive สำหรับเก็บ PDF
1. ไปที่ Google Drive สร้างโฟลเดอร์ใหม่ เช่น `เอกสารบิล Patties Bill`
2. ดับเบิ้ลคลิกเข้าไปในโฟลเดอร์นั้น สังเกตที่ลิงก์ URL ด้านบนของบราวเซอร์ จะเห็นรหัสยาวๆ เช่น `https://drive.google.com/drive/folders/1A2b3C4d5E...`
3. **ก๊อปปี้รหัสยาวๆ หลังคำว่า `folders/` ไว้ (เรียกว่า Folder ID)**

### ขั้นตอนที่ 3: นำโค้ดไปใส่ Google Apps Script
1. ในหน้า Google Sheets ให้ไปที่เมนู **ส่วนขยาย (Extensions) > Apps Script**
2. นำโค้ดจาก `Code.gs` ใหม่ ไปวางทับคใน `Code.gs`
3. นำโค้ดจาก `Index.html` ใหม่ ไปวางในไฟล์ `Index.html`
4. กดบันทึก (Save/รูปแผ่นดิสก์)

### ขั้นตอนที่ 4: Deploy เพื่อขอสิทธิ์ (Authorization) และใช้งาน
1. กดที่ปุ่มสีน้ำเงินมุมขวาบน **การทำให้ใช้งานได้ (Deploy)** > การทำให้ใช้งานได้รายการใหม่ (New deployment)
2. เลือกประเภทเป็น **เว็บแอป (Web app)** ให้เข้าถึงได้ `ทุกคน (Anyone)` หรือ `เฉพาะฉัน` ก็ได้
3. **[สำคัญ]** เวลากด Deploy ระบบของ Google จะเด้งหน้าต่างมาขอสิทธิ์เข้าถึง Google Drive ให้คุณกดเลือก **อีเมลของคุณ -> ขั้นสูง (Advanced) -> ไปที่... (Go to...) -> และกด อนุญาต (Allow)**
4. เมื่อได้ลิงก์ Web App URL ให้นำไปเปิดใช้งานได้เลย

### ขั้นตอนที่ 5: เริ่มใช้งานครั้งแรกบนเว็บ
1. เปิดหน้าเว็บจากลิงก์ที่ได้
2. ไปที่เมนู **ตั้งค่าระบบ (Settings)** มุมซ้ายล่าง
3. **นำ Folder ID** (ที่ก๊อปปี้ไว้ในขั้นตอนที่ 2) มาวางแล้วระบบกดปุ่ม "บันทึก"
4. เสร็จสิ้น! คุณสามารถลองสร้างบิล แล้วกดปุ่ม **"สร้างเป็น PDF บน Drive"** ได้เลยครับ!
