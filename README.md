# ระบบจัดการบิล Patties Bill (Modern Web App)

ระบบใบแจ้งหนี้เวอร์ชันนี้ได้รับการอัปเกรดให้เป็น **Web Application แบบหน้าเดียว (SPA - Single Page Application)** ที่สวยงาม ทันสมัย และมีฟังก์ชันครบถ้วนสำหรับบริหารจัดการบิล ทั้งค่าไฟ ค่าเช่า หรือค่าใช้จ่ายใดๆ สำหรับผู้พักอาศัย

## ฟังก์ชันเด่นที่เพิ่มเข้ามา (Features)
- 🔍 **ระบบค้นหา (Search):** สามารถค้นหาบิลได้อย่างรวดเร็ว จากชื่อลูกค้า, เบอร์โทร, ห้อง หรือ Log No.
- 📝 **เพิ่ม/ลบ/แก้ไข (CRUD):** ไม่ต้องไปแก้ใน Google Sheet ด้วยตัวเองอีกต่อไป สามารถกดปุ่ม สร้างบิล, แก้ไขบิลเก่า หรือลบบิล ได้จากหน้าเว็บโดยตรง
- 🧮 **คำนวณและแปลงยอดเงินอัตโนมัติ:** เมื่อกรอกตัวเลขค่าเช่า/ค่าน้ำไฟ ระบบจะรวมยอด และ *แปลงตัวเลขนั้นเป็นตัวอักษรภาษาไทย (เช่น หกพันบาทถ้วน)* ให้อัตโนมัติ!
- 🖨️ **รูปแบบบิลที่เป๊ะเหมือนเดิม:** เมื่อกดพิมพ์บิล (Print) จะเข้าสู่เทมเพลตบิลดั้งเดิมของคุณ (รูปแบบ 2 ส่วน บน-ล่าง สำหรับฉีก) เพื่อนำไปใช้งานได้อย่างสวยงามเหมือนเดิม
- 💾 **Google Drive Database:** บันทึกข้อมูลและประวัติทั้งหมดลงใน Google Sheets ของคุณแบบเรียลไทม์ ปลอดภัย และฟรี

## 🚀 วิธีการติดตั้งและเริ่มใช้งาน

### ขั้นตอนที่ 1: เตรียม Google Sheets ของคุณ
1. ไปที่ Google Sheets และเพิ่มคอลัมน์ใหม่ที่แถวที่ 1 ให้ตรงตามนี้ (ห้ามเว้นว่างตัวสะกด):
   **`ID`**, **`LogNo`**, **`Date`**, **`Room`**, **`CustomerName`**, **`CustomerPhone`**, **`Item1Desc`**, **`Item1Amt`**, **`Item2Desc`**, **`Item2Amt`**, **`Item3Desc`**, **`Item3Amt`**, **`Item4Desc`**, **`Item4Amt`**, **`TotalAmt`**, **`TotalText`**, **`QRPicUrl`**, **`BankName`**, **`AccName`**, **`AccNo`**
2. *หมายเหตุ:* ถ้ามีข้อมูลเดิมอยู่แล้ว แค่แทรกคอลัมน์ `ID`, `CustomerName`, `CustomerPhone` เข้าไป เพิ่มเติมก็พอ

### ขั้นตอนที่ 2: นำโค้ดไปใส่ Google Apps Script (Backend)
1. ในหน้า Google Sheets ให้ไปที่เมนู **ส่วนขยาย (Extensions) > Apps Script**
2. ลบโค้ดเดิมใน `Code.gs` และก๊อปปี้โค้ดจากไฟล์ `Code.gs` ที่ผมให้ไว้ใหม่ในโฟลเดอร์นี้ไปวาง
3. เอาโค้ดจาก `Index.html` (ที่มีหน้าตาสวยๆ และระบบค้นหา/แก้ไขใหม่) ตรงเข้าไปแทนที่ในไฟล์ `Index.html` ของคุณ
4. กดบันทึก (Save/รูปแผ่นดิสก์)

### ขั้นตอนที่ 3: Deploy เพื่อใช้งาน (เผยแพร่)
1. กดที่ปุ่มสีน้ำเงินมุมขวาบน **การทำให้ใช้งานได้ (Deploy)** > การทำให้ใช้งานได้รายการใหม่ (New deployment)
2. เลือกประเภทเป็น **เว็บแอป (Web app)**
3. ตั้งค่าการเข้าถึง: **ทุกคน (Anyone)** หรือ **เฉพาะฉัน (Only myself)** ตามต้องการ
4. กด Deploy แล้วคุณจะได้ **URL ของหน้าเว็บคุณ** ให้นำลิงก์นั้นไปเซฟไว้ใช้ในมือถือ หรือเปิดในคอมพิวเตอร์เพื่อเข้าไปบริหารจัดการบิลได้ตลอดเวลาครับ!

---

*คำแนะนำระหว่างพัฒนา: หากคุณเปิดไฟล์ `Index.html` จากในเครื่องตรงๆ (เป็นไฟล์ Local .html) ระบบจำถูกจำลองข้อมูล (Mock Data) ขี้นมาเพื่อให้คุณทดสอบดูหน้าตา UI และการคำนวณต่างๆ ได้ทันทีโดยไม่ต้องเชื่อมเน็ตครับ*
