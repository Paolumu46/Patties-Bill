# ระบบจัดการบิล Patties Bill (Modern Web App)

ระบบใบแจ้งหนี้เวอร์ชันนี้ได้รับการอัปเกรดให้เป็น **Web Application แบบหน้าเดียว (SPA)** ที่มีระบบ **ยืนยันตัวตนผู้ใช้งานด้วย Google Account (Authentication)** นอกเหนือจากการจัดการบิลและ Google Drive ครับ

## ฟังก์ชันเด่นที่เพิ่มเข้ามาล่าสุด (Features)
- 🔐 **ระบบตรวจสอบสิทธิ์เข้าถึง (Login/Auth System):**
  - เมื่อเปิดหน้าเว็บ ระบบจะเช็ค **"อีเมลบัญชี Google"** ปัจจุบันที่เข้าใช้งานทันที (ไม่ต้องมีรหัสผ่าน เพราะผูกกับ Google แบบปลอดภัย 100%)
  - หน้าเว็บจะแสดงอีเมลของผู้ใช้อยู่ที่เมบาร์ด้านซ้าย
  - **จำกัดสิทธิ์การใช้งาน:** เฉพาะ "เจ้าของระบบ" หรือ "อีเมลที่ได้รับอนุญาต" เท่านั้นถึงจะเข้าดูบิลแก้ไขบิลได้ คนอื่นเปิดมาจะเจอหน้า "ปฏิเสธการเข้าถึง"
- 💾 **ระบบจัดการไฟล์ PDF (Google Drive Integration):**
  - สร้างและบันทึกบิลเป็นไฟล์ **PDF** ลงใน Google Drive โดยตรง
  - มีลบกและดูไฟล์ PDF ล่าสุดในเมนู File Manager

## 🚀 วิธีการติดตั้งและส่งมอบสิทธิ์ให้ผู้ใช้งานอื่น

### ขั้นตอนที่ 1: Deploy ระบบ
1. นำโค้ด `Code.gs` และ `Index.html` ไปอัปเดตใน Apps Script ของคุณ
2. ไปที่ **การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้รายการใหม่**
   - **เรียกใช้ในฐานะ (Execute as):** เลือกว่า "ฉัน (อีเมลของคุณ)" 
   - **ผู้ที่มีสิทธิ์เข้าถึง (Who has access):** เลือก **"ทุกคนที่มีบัญชี Google (Anyone with Google Account)"** 
3. แบบนี้ใครที่มีลิงก์เว็บ ก็จะถูกระบุตัวตนด้วยอีเมล Google ของตัวเองได้

### ขั้นตอนที่ 2: การตั้งค่าผู้มีสิทธิ์ใช้งานระบบ
1. ในฐานะ **"เจ้าของระบบ"**, ให้เปิดเว็บแอปจากลิงก์ที่ Deploy เสร็จแล้ว
2. ไปที่เมนู **การตั้งค่าระบบ (Settings)**
3. ในช่อง "ระบบกำหนดสิทธิ์การเข้าถึง (Allowed Google Emails)" ให้ใส่ **อีเมลของพนักงานหรือผู้จัดการคนอื่น** ที่อนุญาตให้แก้บิลได้
   - ตัวอย่าง: `manager_a@gmail.com, staff_b@gmail.com`
   - *หมายเหตุ: อีเมลเจ้าของที่เป็นคนสร้างระบบ ไม่ต้องใส่ในช่องนี้ เพราะเข้าได้เสมอ*
4. กดบันทึก ถ้าอีเมลไหนไม่ได้อยู่ในช่องนี้ จะไม่สามารถเข้าหน้าเว็บนี้ได้เลยครับ! ปลอดภัย 100%
